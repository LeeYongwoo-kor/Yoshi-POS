Welcome to Oi-POS!
---
飲食店にてQRコードでの注文を可能にするシステムです。POSシステムでは、顧客のQRコード注文を管理することができます。

既存のQRコード注文 + POSシステムも多数ありますが、アプリではなく、Web画面ですべてを網羅できることが特徴です。
